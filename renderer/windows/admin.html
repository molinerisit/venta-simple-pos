Hardware y Periféricos

GUÍA: ¿El escáner o la impresora no funcionan?

Configurar el hardware correctamente es clave. Sigue estos pasos si encuentras problemas.

Configuración del Escáner de Códigos de Barras

  1. Modo Emulación de Teclado (Keyboard Wedge): La mayoría de los escáneres USB funcionan "de caja" simulando ser un teclado. Si tu escáner no escribe el código en un bloc de notas, es probable que no esté en este modo.
  2. Configuración Inicial: A veces, el lector de códigos de barras necesita una configuración inicial para funcionar correctamente (ej: para leer ciertos tipos de códigos, añadir un "Enter" al final, etc.).
  3. ¿Cómo se configura?: Esta configuración se realiza escaneando una serie de códigos de barras especiales que vienen en el manual de usuario del dispositivo.
    • Busca el manual físico que vino con tu escáner.
    • Si no lo tienes, busca en internet el manual en PDF con el nombre completo, marca y modelo de tu lector (ej: "Manual PDF Symbol LS2208").
    • Dentro del manual, busca secciones como "Configuración de Interfaz", "Interface Settings" o "Suffix Settings" para encontrar los códigos de barras de configuración.

Configuración de la Impresora de Tickets (Ticketera)

  1. Instalar los Drivers: Este es el paso más importante. Antes de que el sistema pueda ver la impresora, debes instalar el software (drivers) que el fabricante proporciona.
    • Busca en internet "drivers para [Marca y Modelo de tu impresora]" (ej: "drivers Epson TM-T20II").
    • Descarga el instalador desde la página oficial del fabricante y ejecútalo.
  2. Reiniciar la PC: Después de instalar los drivers, es muy importante reiniciar tu computadora para que los cambios se apliquen correctamente.
  3. Seleccionar en el Sistema: Una vez reiniciada la PC, vuelve a esta pantalla, haz clic en el botón "Refrescar Puertos" y tu impresora debería aparecer en la lista desplegable "Impresora de Tickets".
  4. Guardar la Selección: No olvides seleccionarla en la lista y hacer clic en "Guardar Hardware".
Generalmente no es necesario para escáneres USB que emulan un teclado.
Asegúrate de haber instalado los drivers primero.

Credenciales de Mercado Pago

GUÍA PASO A PASO: ¿Cómo obtener mis credenciales?

Configurar Mercado Pago es un proceso que se hace una sola vez. Sigue estos pasos con atención.

Paso 1: Obtener Credenciales Básicas

  1. Haz clic aquí para ir al Panel de Desarrolladores de Mercado Pago e inicia sesión.
  2. Haz clic en "Crear aplicación". Dale un nombre (ej: "Mi Sistema TPV La Macelleria") y acepta los términos.
  3. Dentro de tu nueva aplicación, ve a la sección "Credenciales de Producción" y actívalas.
  4. Copia los siguientes dos datos y pégalos en los campos de abajo:
    • El Access Token (empieza con APP_USR-...)
    • El User ID (un número largo, es tu ID de vendedor)

Paso 2: Activar Permisos de la API (¡Muy Importante!)

  1. En la misma página de tu aplicación, busca la sección "Detalles de aplicación" y haz clic en el botón azul "Editar".
  2. Desplázate hacia abajo hasta encontrar la sección "Permisos de la aplicación".
  3. Marca las casillas de los siguientes permisos. Son necesarios para que el sistema pueda crear y gestionar cobros:
    • ✅ write_stores (para administrar tiendas)
    • ✅ pos_write (para administrar cajas)
    • ✅ offline_access (para que la conexión no expire)
    • ✅ payments (para gestionar pagos)
  4. Al final de la página, haz clic en el botón "Guardar cambios".

Paso 3: Renovar Token y Cargar Cajas

  1. Vuelve a la pestaña de "Credenciales de Producción".
  2. Haz clic en el botón "Renovar mis credenciales". Esto es obligatorio para que tu Access Token obtenga los nuevos permisos.
  3. Copia el NUEVO Access Token que aparecerá y pégalo de nuevo en el campo "Access Token" de abajo, reemplazando el viejo.
  4. Con las credenciales actualizadas, haz clic en el botón de abajo que dice "Verificar y Cargar Cajas". El sistema se conectará a Mercado Pago y listará tus cajas disponibles.

Paso 4: Guardar Configuración Final

  1. Una vez cargadas, selecciona la caja que deseas usar en este sistema desde el menú desplegable.
  2. Finalmente, haz clic en el botón "Guardar Credenciales MP".

¡Listo! Tu sistema ya está configurado para recibir pagos con QR.

Configuración de Facturación AFIP

¿Cómo obtengo los archivos de Certificado y Clave?

Para facturar electrónicamente, necesitas un Certificado Digital y una Clave Privada generados para tu CUIT. Este es un proceso técnico que generalmente realiza un contador o tú mismo siguiendo una guía oficial de AFIP.

Pasos Generales:

  1. Generar Clave Privada y CSR: Usando una herramienta como OpenSSL, se genera un archivo private.key (tu clave, ¡guárdala bien!) y un request.csr (una solicitud).
  2. Enviar CSR a AFIP: Desde el portal de AFIP con tu Clave Fiscal, en el servicio "Administrador de Certificados Digitales", envías el archivo request.csr.
  3. Descargar Certificado de AFIP: AFIP te devolverá un archivo de certificado, generalmente con extensión .crt.

Una vez que tengas los archivos private.key y nombre_de_tu_empresa.crt, debes seleccionarlos en los campos de abajo.

Gestión de Caja (Arqueo)

¿Cómo funciona el Arqueo de Caja?

Al habilitar esta opción, el sistema requerirá que un cajero realice una "Apertura de Caja" antes de poder registrar ventas. En ese momento, deberá ingresar el monto de dinero inicial (cambio).

Al final del turno, el cajero deberá realizar un "Cierre de Caja", donde el sistema calculará los totales y permitirá imprimir un resumen detallado del turno (arqueo).

Los horarios de los turnos son informativos y pueden usarse en el futuro para recordatorios automáticos.

Balanzas Kretz (Wi-Fi / Bluetooth)

Cómo conectar una balanza Kretz
  • Wi-Fi (TCP): configurá la IP y puerto de la balanza (ej: 192.168.0.50:8000).
  • Bluetooth (SPP): emparejá en Windows y copiá la dirección (ej: 00:1A:7D:DA:71:13).
  • El protocolo por defecto es Kretz Report (ASCII).

Sincronización en la Nube y Suscripción

¿Cómo funciona la sincronización y la suscripción?

La sincronización en la nube y las copias de seguridad forman parte de las funciones premium del sistema. Para utilizarlas necesitás una suscripción activa.

¿Qué incluye?

  • Respaldo automático de tus datos en la nube.
  • Sincronización entre múltiples equipos/puestos.
  • Restauración ante fallos o pérdida del equipo.
  • Acceso remoto a catálogos, ventas y estadísticas.

Cómo activarlo (paso a paso)

  1. Contratá una suscripción en www.sitiowebejemplo.com.
  2. Vas a recibir tu Token de Licencia (también visible en tu panel del sitio).
  3. Pegá el token en Token de Licencia y completá la URL de la API (por defecto: https://servidor-api-ventasimple-production.up.railway.app).
  4. Activá Sincronización y presioná Guardar Configuración de Sincronización.
  5. El sistema validará la licencia y mostrará el Estado de la Suscripción debajo.

Notas importantes

  • Se requiere conexión a Internet para validar la licencia y sincronizar.
  • Si pasan 72 horas sin poder validar la suscripción, el sistema puede bloquear las funciones premium hasta reconectar.
  • El Token de Licencia es sensible: no lo compartas ni lo publiques.
  • Si desactivás la sincronización, la app sigue funcionando de forma local (sin funciones premium).
